
Initial Setup:
=============

Environment setup:
-----------------

If using bash on *nix systems, run (some tweaks may be needed):
$ . .setup.sh

And you should get a bash prompt:
(msw/master) $

Where:
  * "msw" stands for MineSweeper.
  * "master" is the current git branch.

This will prepend
 /usr/local/node-v10.13.0-linux-x64/bin/
to the path, so node 10.13.0 and npm 6.4.1 will be used.


Setup your IDE to use:
 - node 10.13.0
 - typescript from node_modules/typescript (v3.2.2).
 - JS: ECMAScript 6 in general
 - ReactJSX in web/ folder.
